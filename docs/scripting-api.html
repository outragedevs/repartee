Scripting — API Reference

Complete reference for the api object passed to every script's setup function.

function setup(api)
    -- api.on, api.irc, api.ui, api.store, api.config, api.log, etc.
end

Events

api.on(event, handler, priority?)

Register an event handler. Returns a handler ID for removal.

local id = api.on("irc.privmsg", function(ev)
    -- handle message
end)

Parameters:

Param Type Description
event string Event name (see event list below)
handler function Handler function receiving an event table (ev)
priority number Optional. Default: 50 (api.PRIORITY_NORMAL)

Handlers run in descending priority order. Return true from a handler to suppress the event (prevent lower-priority handlers and built-in handling from running).

api.once(event, handler, priority?)

Same as api.on() but the handler fires only once, then removes itself.

api.off(id)

Remove a previously registered handler by its ID.

local id = api.on("irc.privmsg", handler)
api.off(id)  -- remove the handler

Priority Constants

Constant Value
api.PRIORITY_HIGHEST 100
api.PRIORITY_HIGH 75
api.PRIORITY_NORMAL 50
api.PRIORITY_LOW 25
api.PRIORITY_LOWEST 0

IRC Events

These events fire when the IRC server sends data. All IRC events include a connection_id field.

irc.privmsg

A channel or private message.

api.on("irc.privmsg", function(ev)
    -- ev.connection_id  (string)
    -- ev.nick           (string)
    -- ev.ident          (string)
    -- ev.hostname       (string)
    -- ev.target         (string) channel name or your nick
    -- ev.channel        (string) same as target for channels, nil for PMs
    -- ev.message        (string)
    -- ev.is_channel     (boolean)
end)

irc.action

A CTCP ACTION (/me message). Same fields as irc.privmsg.

irc.notice

Field Type Description
connection_id string
nick string Sender (nil for server notices)
target string
message string
from_server boolean

irc.join

Field Type Description
connection_id string
nick string
ident string
hostname string
channel string

irc.part

Same as irc.join plus message (part reason).

irc.quit

Field Type Description
connection_id string
nick string
ident string
hostname string
message string Quit reason

irc.kick

Field Type Description
connection_id string
nick string Who kicked
ident string
hostname string
channel string
kicked string Who was kicked
message string Kick reason

irc.nick

Field Type Description
connection_id string
nick string Old nick
new_nick string New nick
ident string
hostname string

irc.topic

Field Type Description
connection_id string
nick string Who changed it
channel string
topic string

irc.mode

Field Type Description
connection_id string
nick string Who set the mode
target string Channel or nick
modes string Mode string (e.g. "+o nick")

irc.invite

Field Type Description
connection_id string
nick string Who invited
channel string
invited string Who was invited

irc.ctcp_request / irc.ctcp_response

Field Type Description
connection_id string
nick string
ctcp_type string e.g. "PING", "TIME"
message string

irc.wallops

Field Type Description
connection_id string
nick string
message string
from_server boolean

Lifecycle Events

connected

Fired when a connection to an IRC server is established.

Field Type
connection_id string
nick string

disconnected

Fired when a connection is lost.

Field Type
connection_id string

command_input

Fired before a slash command executes. Return true to suppress the command.

Field Type
command string
args table
connection_id string

Custom Commands

api.command(name, def)

Register a custom slash command.

api.command("greet", {
    handler = function(args, connection_id)
        local target = args[1] or "world"
        api.irc.say(target, "Hello, " .. target .. "!")
    end,
    description = "Send a greeting",
    usage = "/greet <nick>",
})

Parameters:

Field Type Description
handler function Called with (args, connection_id). args is a table of strings.
description string Shown in /help
usage string Usage string shown in help

Commands registered by a script are automatically removed when the script is unloaded.

IRC Methods — api.irc.*

api.irc.say(target, message)

Send a PRIVMSG to a channel or nick.

api.irc.action(target, message)

Send a CTCP ACTION (/me).

api.irc.notice(target, message)

Send a NOTICE.

api.irc.join(channel, key?)

Join a channel, optionally with a key.

api.irc.part(channel, message?)

Leave a channel with an optional part message.

api.irc.raw(line)

Send a raw IRC protocol line.

api.irc.nick(new_nick)

Change your nickname.

api.irc.whois(nick)

Send a WHOIS query.

api.irc.mode(channel, mode_string)

Set a channel or user mode.

api.irc.kick(channel, nick, reason?)

Kick a user from a channel.

api.irc.ctcp(target, type, message?)

Send a CTCP request.

UI Methods — api.ui.*

api.ui.print(text)

Display a message in the active buffer.

api.ui.print_to(buffer_id, text)

Display a message in a specific buffer.

api.ui.switch_buffer(buffer_id)

Switch to a buffer by its ID.

api.ui.execute(command_line)

Execute a slash command (e.g. api.ui.execute("/set theme default")).

State Access — api.store.*

api.store.our_nick(conn_id?)

Returns your current nick, or nil if not connected. If conn_id is omitted, uses the active connection.

api.store.active_buffer()

Returns the active buffer ID, or nil.

api.store.connections()

Returns a table of all connection IDs.

api.store.connection_info(id)

Returns a table with details about a connection (server, nick, connected status, etc.).

api.store.buffers()

Returns a table of all buffer IDs.

api.store.buffer_info(id)

Returns a table with details about a buffer (name, type, topic, connection_id, etc.).

api.store.nicks(buffer_id)

Returns a table of nicks in a channel buffer.

Config — api.config.*

api.config.get(key, default?)

Get a per-script config value. Returns default (or nil) if the key is not set.

local timeout = api.config.get("timeout", 300)

api.config.set(key, value)

Set a per-script config value at runtime.

api.config.set("timeout", 600)

api.config.app_get(key_path)

Read an app-level config value using dot-path notation.

local theme = api.config.app_get("general.theme")
local nick = api.config.app_get("connections.libera.nick")

Timers

api.timer(ms, handler)

Start a repeating timer. Returns a timer ID.

local id = api.timer(60000, function()
    api.ui.print("One minute has passed")
end)

api.timeout(ms, handler)

Start a one-shot timeout. Returns a timer ID.

api.timeout(5000, function()
    api.ui.print("5 seconds elapsed")
end)

api.cancel_timer(id)

Cancel a timer by its ID.

local id = api.timer(1000, tick)
api.cancel_timer(id)

Logging

api.log(message)

Log a debug message. Output is visible when scripts.debug = true in config.

api.log("processing message from " .. ev.nick)
